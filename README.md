# dotfiles

For keeping dotfiles.

to setup, run
```
git clone https://github.com/skasai5296/dotfiles.git --recursive
cd dotfiles
./setup.sh
```
then install Vundle plugins with
```
vim
:PluginInstall
:qa
```
and install `.bash_it` with 
```
.bash_it/install.sh
```

dotfiles include
- `.ssh`
- `.vim`
- `.vimrc`
- `.tmux.conf`
- `.tmux.reset.conf`
- `.bashrc`
- `.bash_profile`
- `.bash_it`
- `.inputrc`
