# Config and setup (mainly for macOS)


## Dotfiles setup

Run
```
bash -c "$(curl -fsSL https://raw.githubusercontent.com/skasai5296/dotfiles/master/setup.sh)"
```

dotfiles include
- `.ssh`
- `.vim`
- `.vimrc`
- `.tmux.conf`
- `.tmux.reset.conf`
- `.bashrc`
- `.bash_profile`
- `.bash_it`
- `.inputrc`
